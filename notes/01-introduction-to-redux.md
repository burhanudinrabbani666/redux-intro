# Introduction to REDUX

- 3rd-party library to manage global state
- **Standalone** library, but easy to integrate with react apps using react-redux library
- All global state is stored in one globally accessiblestore, which is easy to updte using "actions" like useReducer
- its conceptually similiar to using the Context API + useReducer
- Two "versions":
  1. Classic redux,
  2. Modern redux toolkit
